# GitHub Pages kezdőcsomag

## Lépések
1. Hozz létre egy publikus repo-t a GitHubon.
   - **Felhasználói oldal**: `felhasznalonev.github.io`
   - **Projekt oldal**: tetszőleges név (pl. `honlapom`)

2. Töltsd fel ezt a csomagot (főként az `index.html`-t).

3. Engedélyezd a GitHub Pages-t a **Settings → Pages** menüben.
   - Source: `Deploy from a branch`
   - Branch: `main` → `/ (root)`

4. (Opcionális) Saját domainhez adj a repo gyökerébe egy `CNAME` fájlt a domainneveddel (pl. `www.pelda.hu`).

## Használat
- Felhasználói oldal URL: `https://felhasznalonev.github.io`  
- Projekt oldal URL: `https://felhasznalonev.github.io/honlapom/`
